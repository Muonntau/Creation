# Angel RPG

This is a simple text-based RPG where you play as an angel on Earth. Complete
tasks to earn karma while managing your limited energy. Rest when tired,
purchase items in the heavenly store, and level up as you spread goodwill.

## Running the Game

Run the game using Python 3:

```bash
python3 main.py
```

You'll be prompted for your angel's name and then presented with tasks. Choose
tasks by entering their number, rest with `r`, open the store with `s`, or quit
by typing `q`.
